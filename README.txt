This unity project uses unity version 2019.4.0f1 with 2D extras.

To install 2D extras:
	Download the zip from https://github.com/Unity-Technologies/2d-extras
	Unzip the folder into the packages folder
	Rename the extras folder com.unity.2d.tilemap.extras
	In the Packages/manifest.json add "com.unity.2d.tilemap.extras": "https://github.com/Unity-Technologies/2d-extras.git#master"
